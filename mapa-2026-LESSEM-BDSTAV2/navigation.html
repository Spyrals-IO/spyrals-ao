📚 À propos de ces maquettes

Ces maquettes HTML interactives démontrent l'interface du projet StationoScope V2 pour l'INRAE LESSEM. Elles illustrent les fonctionnalités clés décrites dans le cahier des charges : webmapping avec MapLibre, fiches descriptives avec graphiques interactifs, et système de navigation territorial.

Note : Les données affichées sont fictives et générées pour la démonstration. L'application finale utilisera les vraies données de la base BDSTAV2.
